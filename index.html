About Nav

Let me introduce myself, my name is Nav Appaiya and I am a PHP Developer living in The Netherlands. I was born in Jaffna, Sri Lanka in the year 1992 and have migrated with my parents to the Netherlands in 1994.

Software and telecommunication has been with me since I was a kid. I remember myself disassembling phones and computers from a early age. I have kept myself busy in the IT and have worked for several companies in the Netherlands.

Mijn werk Open-source

Project 'ZippyTheUnzipper' (PHP & HTML)

ZippyTheUnzipper is een kleine applicatie die ik geschreven heb voor het online uitpakken van bestanden. Zippy heb ik geschreven in de tijd dat ik grote bestanden over moest zetten van webhosters. Het downloaden/uploaden van losse bestanden naar een webhosting is niet altijd ideaal, en helemaal niet als je tijdens een transfer onderbroken word. Met Zippy kun je je bestanden zippen, en vervolgens 1 bestand uploaden naar je webhost, die je met zippy snel en eenvoudig kan uitpakken.

Project NavWunderlistBundle (Symfony 3 Bundle)

This is a Symfony 2 Wunderlist bundle which helps you integrate your Wunderlist Tasks in you Symfony application. It uses the Wunderlist API to connect to your account and retrieve all tasks from there. Also you can create, update or delete tasks through this Symfony bundle.

Project TwitterBot (Symfony 3)

This is a Symfony3 with Guzzle project. This simple twitterbot that can tweet interesting tech releated postst to twitter. It can be configured to trigger a unique tweet each time it gets hit by a special url. It is built with Symfony 3 and GuzzleHTTP 3.

Check this out

Project Tradetracker (Symfony 3)

This is a small integration bundle that I wrote to encorporate TradeTracker into Symfony. This has no direct use, but has alot of configuration setup to start working with Tradetracker through their API. You can finde some examples in the project how I used them, and also there are some entities provided for products that you can harvest from the API.

Check this out

Project WooCommerce Bundle (Symfony)

A small project that I did to make my own life easier when working with Woocommerce inside my Symfony projects. This bundle can integrate external woocommerce webshops into Symfony with the Woo-API. Also there is support for multiple shops, and the WC-API version 2 is used here. Not longer maintained, but you can find it on github and use it if you like.
